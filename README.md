# si5-prog-web-project
## Backend repository

## How to start the project :
```bash
$ cd app/main
$ docker-compose up -d  
$ npm i 
$ npm start
```

## You can see the api documentation on :
```text
GET localhost:8080/api
```

## How to populate backend remotely :
```text
POST localhost:8080/api/sales-points?method=remote
```
